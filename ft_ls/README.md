# FT_LS Project

## Project Overview
FT_LS is a 42 Project that aims to mimic libc ls function

* Basic functions:
1) -l, -R, -a, -r, -t.

* Bonus:
1) -c, -1, -d.

## Sources
* http://man7.org/linux/man-pages/man1/ls.1.html
* http://stackoverflow.com/questions/4674006/set-specific-bit-in-byte

## Installation
```
$> make
$> ./ft_ls {optional argument}
```
