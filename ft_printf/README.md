# PRINTF Project

## Project Overview
PRINTF is a 42 Project that aims to mimic libc printf function

## Sources
* https://linux.die.net/man/3/printf

## Installation
```
$> make
$> gcc libftprintf.a srcs/main.c && ./a.out
```
