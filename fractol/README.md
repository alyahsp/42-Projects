# Fractol Project

## Summary
<img align="center" src="http://image.noelshack.com/fichiers/2016/52/1483109173-screen-shot-2016-12-30-at-3-40-16-pm.png" width="100%" />
Fractol is a 42 Project aimed at exploring fractals; mathematical sets that exhibit a repeating pattern displayed at every scale.

## Functions
* Increase or decrease current iteration.
* Zoom or unzoom.
* Different colors patterns.
* Random color patterns with random variable not using time stamp.
